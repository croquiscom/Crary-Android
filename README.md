## Installation
### 1. include repository
``` groovy
    repositories {
        maven {
            url 'https://github.com/croquiscom/Crary-Android/raw/master/deploy/releases'
        }
    }
```

### 2. dependency
``` groovy
    compile 'com.croquis.crary:crary:0.2.3@aar'
```

## Build
```
    gradle clean build uploadArchives
```
